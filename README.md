## Description
Morning Glory Cafe Order System is a Python Tkinter-based desktop application
designed to manage cafe orders efficiently using an SQLite database.

## Features
- Add new cafe orders
- View all orders
- Update existing orders
- Delete orders
- SQLite database integration
- User-friendly Tkinter GUI

## Technologies Used
- Python
- Tkinter
- SQLite
